# campus-network-design-cisco
A comprehensive campus network implementation featuring VLAN segmentation, Inter-VLAN routing, and secured network services.

📌 Project Overview
This project demonstrates a fully functional enterprise network designed in Cisco Packet Tracer. It simulates a multi-department campus environment with automated services and strict security policies.

🛠 Features & Protocols
VLANs (Virtual LANs): Segmented network traffic for Departments (e.g., HR, Finance, IT).
Inter-VLAN Routing: Enabled communication between departments using a Router-on-a-Stick configuration.
DHCP Server: Centralized IP management to automate device addressing across all VLANs.
DNS Server: Implemented for local domain name resolution.
ACLs (Access Control Lists): Applied security policies to restrict unauthorized access between sensitive departments.



🚀 How to Run
Download the .pkt file from this repository.
Open it using Cisco Packet Tracer (v8.x or higher).
Test connectivity by pinging between devices or accessing the DNS-configured web server.

**Note:**
After starting wait for a minute for the page to refresh and the ports to open.
When pinging note that the first packet would always fail and then try the second packet using the same and it will be successfull.
